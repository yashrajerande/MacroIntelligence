# MacroIntelligence
This will manage the Macro Dashboard that i want to publish daily.

## Departments
- **MacroIntelligence Corp** (daily) — multi-agent macro dashboard. See
  `macro-intelligence/README.md`.
- **ConglomeratesTracker** (monthly, 1st of every month) — strategic-power
  report on India's top 21 conglomerates. Publishes to GitHub Pages as a
  tab on the MacroIntelligence root and to Notion. See
  `macro-intelligence/agents/ConglomeratesTracker/README.md`.

The repo root `index.html` is a tab shell — visitors land on Daily Macro
by default and can switch to Conglomerates with one click. Append
`?tab=conglomerates` to deep-link.
