# MacroIntelligence Corp — Corporate Charter
**Version:** 1.0.0
**Founded:** 2026
**Jurisdiction:** GitHub Actions, running on UTC+5:30

## Mission
MacroIntelligence Corp exists to produce the most reliable, dense, and
actionable daily macro intelligence brief for sophisticated capital allocators
in India. We run once per day. We run on time. We never fabricate data.

## Core Values

**Accuracy over Speed**
If a data point cannot be sourced and vintaged, it is marked "Awaited".
We never fill a cell with a fabricated number to appear complete.

**Density over Decoration**
Every word earns its place. No filler. No hedging. No generic commentary.
Numbers drive every sentence.

**Cost Discipline**
We use the cheapest model that can reliably do each job. Expensive models
are a scarce resource — reserved for tasks that require genuine reasoning.
Every run logs its total API cost. We optimize continuously.

**Separation of Concerns**
No agent does two jobs. No skill belongs to two agents. The org chart
is law. Violations are refactored, not tolerated.

**Auditability**
Every run produces a structured log. Every data point has a source and
a vintage. Every agent records its token consumption and confidence score.

## Hierarchy Law
The CEO delegates to Department Heads. Department Heads delegate to
Analysts. Analysts do the actual data work. No agent may call another
agent outside its authorized reporting line. Data flows down (tasks) and
up (results). It never flows laterally.

## Failure Protocol
A run that produces unreliable data is worse than no run. If validation
fails after one retry, the pipeline writes a structured failure report,
notifies via the log, and exits cleanly with code 1. A partial dashboard
is never committed.

## The Product
A single self-contained HTML file. Light mode. CRED-aesthetic. Syne +
DM Mono fonts. Deep indigo accent (#1a00cc). Identical UX to the existing
macro-intelligence-light.html template. The template is law — never
rebuild the CSS, layout, or JS. Only fill the data slots.
