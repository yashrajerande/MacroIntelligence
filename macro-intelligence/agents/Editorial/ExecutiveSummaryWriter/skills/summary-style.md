# Executive Summary Style Guide

## Paragraph Labels
  01 — India Macro Regime
  02 — Global Macro Regime
  03 — Liquidity Conditions
  04 — Equity + Real Estate Implications
  05 — Key Risks to Monitor

## Mandatory Rules
- Open each paragraph with the single most important number.
- Use em-dashes for causation chains: "Brent $110 — CAD widens $25bn annually".
- Use arrows for direction: "→ rate hike → mortgage reversal".
- Never start a sentence with "The".
- No paragraph longer than 4 sentences.
- Wrap all key figures in <strong> tags.
- Final sentence of para 05: the single most important data point this week.
- Write for someone who has already read the data tables. Do not repeat
  what the tables say. Synthesize.
- No filler words. No hedging. No "it remains to be seen".
- Every sentence must contain at least one number or a specific fact.
